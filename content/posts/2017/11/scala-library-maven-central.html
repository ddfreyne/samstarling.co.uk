--- title: Publishing Scala libraries to Maven Central created_at: 2017-11-05 00:00:00 kind: article ---

As part of my recent work on finagle-prometheus, one of the things I wanted to do was to get the library in Maven Central. This makes it more user-friendly, as it means that you don't have to add a new resolver to your project.

However, doing so was pretty tricky, so I thought it'd be worth writing it down for the benefit of others. The process looks a bit like this:

  1. Release the project.
  2. Publish artifacts to Bintray
  3. Have Bintray sync the artifacts to Maven Central

Prerequisites

You'll need:

  1. A Bintray account, with a Maven repository configured.

Release the project

I use the sbt-release plugin for this, with the default settings. It works really well.

At this point, it's also worth thinking about whether you need to cross-build for different Scala versions.
